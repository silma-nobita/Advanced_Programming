Ocean View Resort Room Reservation System
Project Overview

The Ocean View Resort Room Reservation System is a web-based application designed to manage hotel room bookings. It allows authorized staff to add, view, update, and delete reservations, generate invoices, and manage room types and pricing. This system helps streamline reservation management and provides an intuitive interface for staff to access and maintain reservation data.

Features

User Authentication: Secure login/logout mechanism for authorized staff (Admin/Receptionist).

Reservation Management: Create, view, edit, delete reservations with the ability to search by reservation number or guest name.

Billing: Automatically calculate the total bill based on the number of nights and rate per night. Printable invoice generation.

Help Section: Provides searchable help content for system guidance.

Filtering and Search: Filter reservations by reservation number, guest name, contact, or room type.

Technologies Used

Frontend: JSP, HTML, CSS, JavaScript

Backend: Java Servlets, JSP

Database: MySQL (used for storing reservation details and room rates)

Version Control: Git, GitHub
